Get Started with Colabo!

All contents are shared instantly and saved automatically.


Add text/annotation: Add a "text" element and type in the textarea.

Add a 3D Model: Add a "3D Model" element. You may play around with the model inside the element window, or enlarge it to full-screen.

Add an image: Add an "image" element, then clicking on the small "+" button at the upper-left corner of the card to open the interface for file upload.

Move and resize an element: Drag the element card to move around. Drag from the corner to resize.

Bring an element to front: Double click on the element card.

Delete an element: Click on the "x" icon at the upper-left corner of the element card.
